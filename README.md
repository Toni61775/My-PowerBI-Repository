# My-PowerBI-Repository


# 📊 Power BI Data Visualisation Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on building end-to-end data analytics solutions using **Power BI**. The objective was to transform raw retail and sales data into **insightful, interactive reports** that support business decision-making.

The project demonstrates strong capabilities in **data preparation, modelling, and visual storytelling**, turning complex datasets into clear and engaging dashboards.

---

## 🎯 Objectives
- Clean and transform raw datasets for analysis  
- Build interactive and user-friendly Power BI reports  
- Apply DAX calculations to enhance data modelling  
- Communicate insights effectively through data storytelling  

---

## 🛠️ Tools & Technologies
- **Microsoft Power BI (Desktop)**
- **Power Query Editor**
- **DAX (Data Analysis Expressions)**
- Excel / CSV data sources  

---

## 🔄 Data Transformation & Cleaning (Power Query)
- Cleaned and reshaped raw datasets using **Power Query**  
- Handled missing values, removed duplicates, and corrected data types  
- Applied transformations such as filtering, merging, and aggregating tables  
- Created a structured data model ready for analysis  

---

## 📊 Interactive Reporting & Dashboards
- Designed **interactive reports** with intuitive layouts  
- Implemented **slicers and filters** to enable dynamic data exploration  
- Built dashboards allowing users to drill down into key metrics  
- Ensured a seamless user experience through logical navigation  

---

## 🧮 DAX Calculations
- Created **calculated columns and measures** using DAX  
- Developed KPIs such as total sales, averages, growth metrics, and percentages  
- Applied time-based analysis (e.g., trends over time)  
- Enhanced insights through advanced calculations and custom metrics  

---

## 📉 Data Visualisations
Built a range of visuals to represent retail and sales data effectively:

- **Bar Charts** – Comparing product categories, regions, and performance  
- **Line Charts** – Tracking sales trends over time  
- **Pie Charts** – Showing category or market share distribution  
- **Maps** – Visualising geographical sales performance  
- **Cards & KPIs** – Highlighting key business metrics at a glance  

---

## 📖 Data Storytelling
A key focus of this project was presenting data in a **clear, meaningful narrative**:
- Identified key sales trends and performance drivers  
- Highlighted regional and category-based insights  
- Structured reports to guide users through a logical flow of information  
- Enabled stakeholders to make informed, data-driven decisions  

---

## 🧠 Insights & Impact
- Demonstrated how Power BI can uncover **actionable insights from raw data**  
- Showed how interactive dashboards improve **decision-making efficiency**  
- Emphasised the importance of **clean data and strong data modelling**
